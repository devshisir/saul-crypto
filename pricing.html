What is counted as a request on the REST API?

A request is deemed to be a single one if the limit query parameter on endpoint isn’t available, or isn’t used. Otherwise — if the limit query parameter is available and is used — then each of the 100 data points actually returned in response are counted as one request.

How long does it take to integrate CoinAPI?

We will deliver an API key to your email address within a minute after you subscribe. With this key, you can use our API right away. The integration component is up to you: it can take just five minutes for simple projects or several days for most complicated ones. Don't forget to check out our SDK on GitHub.
